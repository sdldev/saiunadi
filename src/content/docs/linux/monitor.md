## uptime
Melihat waktu operasional server
```sh 
uptime
```
## free
Melihat memory terpakai dan tidak terpakai (termasuk swap) server
```sh 
free
```

## htop
***htop*** monitoring realtime CPU, RAM dan proses yang berjalan pada server

```sh
apt install htop
```
```sh
htop
```
